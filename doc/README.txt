This program is the result of following the android development
tutorial at http://developer.android.com/training/index.html - a
tutorial licensed under the Creative Commons Attribution 2.5.

The program allows for the entry, modification, deletion, and display 
of a note containing a subject, date, and content.
It also implements a simple action bar to display the 
top 100 words in the contents field, and an histogram of the top 5 
words used.

The application requires Android 3.0 (API level 11) or above.
