This program is the result of following the android development
tutorial at http://developer.android.com/training/index.html - a
tutorial licensed under the Creative Commons Attribution 2.5.  
All code comes directly from the tutorial.

The program allows for the entry of text, and the display of text in
another view.  It also implements a simple action bar.

Requires Android 3.0 (API level 11) or above.
